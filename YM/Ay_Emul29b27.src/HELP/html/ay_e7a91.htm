Official Emulator Page

Official Emulator Homepage is found on http://bulba.untergrund.net. There are new versions of Emulator, skins and music archives and also technical information about AY-3-8910/12 and YM2149 chips, music file formats, sources, tools and other can be found on the page. You can download YM-Engine package by Sledge Hammer for using YM sound in your own programs. Also, you can download source package squaretone-0.1.tgz by Martin Andersson to compile player on any non-Microsoft platforms. Vortex Tracker II project is powerful editor for editing AY/YM-music on PC, you can write own tunes in PT3-format in Windows. Download last version from this page.
